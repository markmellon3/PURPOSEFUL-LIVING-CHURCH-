Engineering Details

Key Decisions

Zero Backend Server

Firebase Realtime Database eliminates the need for a custom backend. Authentication, data persistence, and real-time sync all handled client-side with security rules.

Timezone-Aware Calculations

All date operations use local date components instead of toISOString() to prevent UTC offset bugs. Critical for Uganda (UTC+3) where entries after midnight appeared on the wrong day.

Duplicate Detection

Real-time duplicate checking on finance entries compares member + type + amount + date to prevent accidental double entries, with animated warning display.

Contact Import Engine

Supports three file formats (vCard, CSV, JSON) with phone-based deduplication and automatic skip for existing members. Parses vCard FN/TEL/EMAIL fields with regex.

Client-Side PDF Generation

Receipts and monthly reports are generated entirely in the browser using jsPDF with html-to-PDF rendering. No server round-trip, instant download, formatted with monospace receipt styling.
